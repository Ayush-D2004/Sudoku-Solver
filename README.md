# Sudoku-Solver
A python based application to solve a 9 X 9 sudoku

## To Run this program
1. Clone the repository ``git clone https://github.com/Ayush-D2004/Sudoku-Solver.git``
2. Navigate to the project directory
3. Run this command ```python sudoku.py```
